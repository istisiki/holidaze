# Holidaze

A holiday calendar visualizer built with React, TypeScript, and Vite.

## Commands

| Command | Description |
|---------|-------------|
| `bun dev` | Start development server |
| `bun run build` | TypeScript check + production build |
| `bun run lint` | Run ESLint |
| `bun run preview` | Preview production build locally |

## Features

- Manual holiday input with date picker
- File upload (`.txt` format: `YYYY-MM-DD: Holiday Name`)
- Year calendar view with holidays highlighted
- Weekend highlighting
- PDF export
- URL-based sharing
